## Quick Start

### 1. Demo Management
```bash
# Start demo controller
cd /workspace/Medical-AI-Assistant/demo/presentations/scripts
python demo_manager.py --demo-type cardiology --stakeholder c-suite
```

### 2. Scenario Selection
```bash
# Launch cardiology scenario
python demo_scenarios.py --specialty cardiology --scenario stemi
```

### 3. Recording
```bash
# Start demo recording
python demo_recorder.py --output cardiology_demo.mp4
```
