# Changelog

All notable changes to the Medical AI Assistant project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project structure and foundation
- Docker-based development and production environment
- FastAPI backend with WebSocket support
- React frontend with TypeScript and Tailwind CSS
- PostgreSQL 17 Alpine database setup
- Redis caching layer
- LangChain v1.0 agent runtime framework
- RAG layer with Chroma vector store
- Safety filter framework
- Audit logging system
- Comprehensive documentation structure
- CI/CD pipeline with GitHub Actions
- Development environment with hot reload
- Database migration system with Alembic
- User authentication and RBAC
- Session management system
- Preliminary Assessment Report (PAR) generation
- Nurse dashboard interface
- Patient chat interface
- EHR connector interface
- Model registry system
- Training pipeline with PEFT/LoRA support
- Demo environment with 7B model
- Security features and PHI protection

### Documentation
- Added comprehensive README with architecture diagrams
- Created CONTRIBUTING.md with contribution guidelines
- Added CODE_OF_CONDUCT.md
- Docker deployment guide
- API documentation structure
- Security and privacy guidelines

### Infrastructure
- Docker Compose setup for multi-container orchestration
- Development and production Docker configurations
- Database initialization scripts
- Health checks for all services
- Volume management for persistence
- Network configuration for service communication
